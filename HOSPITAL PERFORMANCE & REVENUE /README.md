🏥 Hospital Performance & Revenue Dashboard (Microsoft Excel)

📌 Project Overview

The **Hospital Performance & Revenue Dashboard** is an interactive Microsoft Excel dashboard designed to analyze hospital operations, patient trends, financial performance, and doctor efficiency. The dashboard transforms raw hospital data into meaningful insights through data cleaning, Pivot Tables, Pivot Charts, Slicers, and advanced Excel functions.

This project helps hospital administrators monitor key performance indicators (KPIs), identify trends, and make data-driven decisions to improve operational efficiency and patient care.

📊 Dashboard Preview

🎯 Objectives
- Analyze overall hospital performance.
- Monitor patient admissions throughout the year.
- Track operational expenses and billing revenue.
- Evaluate doctor performance.
- Compare insurance providers and medication counts.
- Identify the most common medical conditions.
- Enable interactive filtering using Excel slicers.

 📈 Key Performance Indicators (KPIs)

- 👨‍⚕️ Total Patients
- 🚑 Emergency Patients
- 💰 Total Operational Expense
- 💵 Average Cost per Patient
- 🏥 Most Common Medical Condition
- 👨‍⚕️ Top Performing Doctors

📉 Dashboard Features

📌 Top 10 Doctors
- Displays the top-performing doctors using a Treemap chart.
📌 Monthly Wise Patients
- Shows monthly patient admission trends using a Line Chart.
📌 Disease Wise Billing Amount
- Compares billing amounts across different medical conditions.
📌 Disease Wise Patients
- Displays patient distribution by disease using a Doughnut Chart.
 📌 Insurance Patients vs Medication
- Compares insurance providers with medication counts.
 📌 Doctor Performance Analysis
- Visualizes doctor performance for comparative analysis.
 📌 Interactive Filters (Slicers)
Users can filter the dashboard by:
- Month
- Medical Condition
- Insurance Provider
- Blood Group
